# Day 2 - MCP & Interoperability

## Concepts Learned

### MCP (Model Context Protocol)

MCP is a standard protocol that allows AI agents to connect to tools, databases, APIs, and external systems.

Think of MCP as USB-C for AI.

Agent
↓
MCP
↓
Tools

Benefits:

* Standardized integrations
* Reusable connections
* Reduced development complexity

---

### A2A (Agent-to-Agent)

Allows different AI agents to communicate and collaborate.

Example:

Research Agent
↓
Writer Agent
↓
Reviewer Agent

Agents can delegate tasks and exchange information.

---

### A2UI (Agent-to-User Interface)

Allows agents to generate user interfaces dynamically.

Examples:

* Dashboards
* Forms
* Charts
* Interactive components

Instead of returning only text, agents can create visual experiences.

---

### AP2 (Agent Payments Protocol)

Provides a secure framework for agent-based payments.

Examples:

* Purchasing services
* Paying subscriptions
* Processing transactions

---

### UCP (Universal Commerce Protocol)

Standardizes how AI agents interact with commerce systems.

Examples:

* Online stores
* Product catalogs
* Shopping workflows
