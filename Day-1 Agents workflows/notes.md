# Day 1 - Agent Workflows

## Concepts Learned

### Sequential Workflow

Research Agent → Writer Agent → Reviewer Agent

Tasks execute one after another.

### Parallel Workflow

Research Agent
Market Agent
Competitor Agent

All execute simultaneously.

### Loop Workflow

Agent repeatedly performs actions until a goal is achieved.

### Human-in-the-Loop

Workflow pauses and waits for user approval before continuing.

## Key Learning

The workflow architecture often matters more than the model itself.

A good workflow can outperform a stronger model with a poor workflow.
